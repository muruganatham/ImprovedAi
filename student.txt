--- Q14: explain what is a linked list? ---
Response:
A **linked list** is a linear data structure where elements, called nodes, are stored in non-contiguous memory locations. Each node contains data and a pointer to the next node, allowing dynamic memory allocation and efficient insertions/deletions compared to arrays. Unlike arrays, linked lists don’t require pre-allocated memory and can grow or shrink during runtime. A common use case is implementing **undo functionality** in applications, where each action is stored as a node.
